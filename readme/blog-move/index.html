博客迁移后的第一篇文章（确信）

发表于 2022-05-03 分类于 说点什么 本文字数： 434 阅读时长 ≈ 1 分钟

        Blog迁移的工作算是正式完工了（别问，问就是放弃之前所写了）

        不知道之前的Blog抽了什么疯，总之无法正常访问，再者看了《向上生长》之后又涌现了写阅读笔记的冲动。因此才开始决定迁移博客。

        当然，我这里所说的放弃并非完全的放弃，之前Blog的语法在NexT中很多都无法使用，所以我会对Blog进行一次小规模的更新。而我对此所做出的解释是：“Blog删库，新的Blog重新建立。”

        其实吧，写了Blog又没什么人看，渐渐的也将Blog看淡了，只将它作为记录已经消磨时间的工具。有很多人说，这都什么年代了，哪里还有人写Blog。对此我只是对他笑了笑，并没有多说什么。为什么要写博客？我也不知道，也许就是想写吧。在现在看来，博客的作用似乎愈来愈不明显了。无所谓了，写都写了，不如就坚持下去。

        最后，基本上可以稳定每个月都发一次《向上生长》或新书的阅读笔记。不仅是为了记录，也是为了那句话——“若你不想在身消体灭之际就被世界所遗忘，那就写一些值得一写的东西，或去做一些值得一写的事情”
