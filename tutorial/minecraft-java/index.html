关于Java Minecraft Forge服务器搭建

发表于 2022-05-03 分类于 教程 本文字数： 1.7k 阅读时长 ≈ 3 分钟

注：本文将会讲解有关Minecraft Java服务器搭建的一些问题，其中包括Forge和Fabric服务器搭建，我将默认你所使用的为Windows操作系统，另外，本文不含插件服搭建。

服务端的选择与使用

Minecraft的服务端种类很多，那么我们应该如何去选择并使用它们呢？

原版

Minecraft原版服务端。原版服务端又被称为官服，它的特性与原版相一致，是完全没有经过任何修改的，十分适合基友服联机使用，此外，它在很多地方都有方法下载，这其中包括MCWiki。

下载

下面将演示Minecraft 1.17.1原版服务端的下载方法。

首先我们在MCWiki打开MC Java 1.17.1的词条，此时我们将目光转向右边。

我们会发现下载处包含服务端，我们点击它即可下载。

你也可以在官网下载。注：官网默认为最新游戏版本。

Mod

Forge

Forge服务端，顾名思义，Forge服务端即为Forge所推出的服务端，可以使用Forge Mod。

下载

下面我将演示Minecraft 1.16.5 Forge服务端的下载方法。

首先我们打开Forge的官网，找到目前最新版本

点击Installer即可下载。

注意：我们现在下载的并非服务端，而是Forge的安装文件，部分版本的安装方法可能有些不同，我将会在最后进行讲解

双击运行我们下载的安装文件。

选择Install server

然后选择一个空文件夹，等待安装即可。

Forge服务端安装时间可能较长，其中可能需要手动补全libraries库，我们可以通过查看log来找到所需文件手动下载。建议使用特殊上网方法进行安装，此处不在过多赘述。

小知识

如果不想观看广告的话，可以使用如下方法：

首先，我们右键Installer，选择复制链接得到的结果如下：

https://adfoc.us/serve/sitelinks/?id=271228&url=https://maven.minecraftforge.net/net/minecraftforge/forge/1.16.5-36.1.33/forge-1.16.5-36.1.33-installer.jar

我们将前面的https://***删除，保留后面的内容：

https://maven.minecraftforge.net/net/minecraftforge/forge/1.16.5-36.1.33/forge-1.16.5-36.1.33-installer.jar

此时，我们就可以将它复制到多线程下载器中进行下载，当然，你可以直接复制到浏览器中下载。

Fabric

Fabric服务端，顾名思义，Fabric服务端即为Fabric所推出的服务端，可以使用Fabric Mod。

下载

下载Fabric服务端，同样需要进入Fabric的官网，我同样以目前最新版为例。

我们点击Download installer，无论是.jar后缀还是.exe后缀都不会影响我们接下来的操作。

点击Server，同样选择一个空目录等待安装即可。

如果需要更换Minecraft版本，请点击下拉框选择。

闪退

问题排查

通用

  1. 确认是否安装JDK
  2. 确认是否正确配置环境变量
  3. 确认服务端是否损坏
  4. 确认是否将eula协议设置为true
  5. 如是游戏版本为1.17+，请确认是否正确安装JDK 16
  6. 确认文件名与启动脚本中文件名一致
  7. 确认使用ansi编码保存
  8. 确认各参数之间添加空格

Forge服务端

  1. 确认是否补全libraries库
  2. 确认libraries库是否损坏

另外，端口被占用也会导致Minecraft服务器无法运行。

关于较低版本Forge服务端下载

较低版本的Forge服务端下载使用方式相对与目前较高版本有所不同，在此处我们以Forge 1.12.2最新版举例。

我们进入Forge官网

依次选择1.12—>1.12.2

我们下载目前最新Forge中的Universal文件。同样，上方所说的小知识对该处依旧适用。

我们将下载好的文件放到一个空文件夹，使用.bat脚本运行即可。

同上，Forge服务端安装时间可能较长，其中可能需要手动补全libraries库，我们可以通过查看log来找到所需文件手动下载。建议使用特殊上网方法进行安装。
